EXPERIENCE

Most recent projects

Coded using Django and therefore Python, this web application allows Community Managers, those who form part of the Spanish Red Cross, to store, filter, visualise and update their daily work information. Soon to be implemented nationally in Spain, this application attributes different permits based on the pyramidal structure of the institution, allowing the people in charge to add different buildings of the organisation and its manager. In addition, only managers will be suitable to add and give permissions to the different CM that volunteer in their venue. These permissions allow personalization about which screens and options will be displayed and can be modified by the manager at any time.
Furthermore, this web application retrieves once a month all the data from all the social media profiles that are currently in use by the Red Cross on Instagram, Facebook and Twitter, such as number of comments, likes, visualizations or more advanced data as gender demographics, users by country and city. All this stored information is later displayed by interactive graphs, allowing them to create better content strategies.

Django, Python, HTML5, CSS3, JavaScript, AJAX, Docker, PostgreSQL, Plotly Dash, Tweepy, Instagram for Developers, Facebook for Developers.

Coded using Django and Python, this web application developed for the Spanish Space Weather Service (SeNMEs) in both Spanish and English, shows information retrieved from different files (CSV and images) provided that get updated every few minutes. It also contains a form to get in touch.
Despite all the back-end and front-end code was made from scratch, the design (name of the sections and data displayed in each of them) was given.

Django, Python, HTML5, CSS3, JavaScript, AJAX, Docker, PostgreSQL.

Coded using Django and Python, this web application developed for the Spanish NHS allows registered users to get access into different graphs. These graphs, implemented using Plotly Dash, show past information of the number of people who got admitted and discharged from hospitals during a period of time, as well as predictions of the number of people who will get admitted within the next week and/or month.
Therefore, using values included through the settings section such as number of patients per nurse and number of nurses per shift, hospital managers are given the information to plan for the month ahead.


Django Volt Dashboard, Python, HTML5, CSS3, JavaScript, AJAX, Docker, PostgreSQL, Plotly Dash.
